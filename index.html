üìñ CCST Glossary

A - C

  • Accuracy: Closeness of measurement to true value, expressed as % of span
  • Actuator: Device that converts signal/energy into mechanical motion
  • Analog Signal: Continuously variable signal (e.g., 4-20mA, 3-15 PSI)
  • As-Found/As-Left: Calibration data before and after adjustments
  • Bias: Constant offset added to controller output
  • Calibration: Comparing and adjusting instrument to known standard
  • Cascade Control: Output of one controller is setpoint for another
  • CV (Flow Coefficient): GPM of water at 1 PSI pressure drop

D - H

  • Damping: Filtering to reduce noise, slows response time
  • Deadband: Range where input change produces no output change
  • Dead Time: Delay between controller action and process response
  • Derivative (D): PID term that anticipates future error based on rate of change
  • DP (Differential Pressure): Pressure difference between two points
  • Drift: Slow change in output over time without input change
  • Dry Leg: DP transmitter side filled with air/gas (not process fluid)
  • Fail Safe: Instrument position on power/air failure (FC, FO, FL)
  • Field Device: Instrument mounted in process area (not control room)
  • Foundation Fieldbus: All-digital communication protocol, multi-drop
  • HART: Highway Addressable Remote Transducer (digital on 4-20mA)
  • Hysteresis: Different output for same input depending on direction

I - O

  • I/P Converter: Current (4-20mA) to Pressure (3-15 PSI) converter
  • Integral (I): PID term that eliminates steady-state error (offset)
  • Intrinsically Safe: Energy-limited circuit safe in hazardous areas
  • LRV (Lower Range Value): Minimum calibrated value (0%)
  • Loop: Complete control circuit from sensor to controller to final element
  • Multidrop: Multiple devices on single wire pair (HART/Fieldbus)
  • Offset: Steady-state difference between setpoint and process variable

P - R

  • P&ID: Piping & Instrumentation Diagram
  • PID: Proportional-Integral-Derivative controller
  • Positioner: Device that ensures valve position matches signal
  • Proportional (P): PID term providing immediate response to error
  • Proportional Band: Input change causing 100% output change
  • PV (Process Variable): The measured value
  • Rangeability: Ratio of max to min measurable flow
  • Repeatability: Consistency of measurements under identical conditions
  • Response Time: Time for output to reach 63.2% of final value
  • RTD: Resistance Temperature Detector (Pt100, Pt1000)

S - Z

  • Setpoint (SP): Desired process variable value
  • SIL (Safety Integrity Level): Reliability of safety system (1-4)
  • Smart Transmitter: Microprocessor-based with digital communication
  • Span: Difference between URV and LRV
  • Stiction: Static friction causing valve sticking/jumping
  • Thermocouple: Temperature sensor using dissimilar metal junction
  • Transmitter: Device converting sensor signal to standard output
  • Turndown: Ratio of normal max flow to min controllable flow
  • Two-Wire: Transmitter powered by same wires carrying signal
  • URV (Upper Range Value): Maximum calibrated value (100%)
  • Wet Leg: DP transmitter side filled with process fluid
  • Zero: Output when measured variable is at minimum
  • Ziegler-Nichols: PID tuning method using ultimate gain/period

Advanced PID Tuning Methods ‚ñº
Ziegler-Nichols Ultimate Gain Method:
1. Set I and D to zero (P-only control)
2. Increase Kp until continuous oscillation occurs
3. Record Ku (ultimate gain) and Tu (oscillation period)
4. Calculate: Kp=0.6Ku, Ti=0.5Tu, Td=0.125Tu
Best for: Quarter-wave decay response

Cohen-Coon Method:
‚Ä¢ Based on process reaction curve (open-loop step test)
‚Ä¢ Measures dead time (Œ∏) and time constant (œÑ)
‚Ä¢ Better than Z-N for processes with significant dead time
‚Ä¢ Provides aggressive tuning for load disturbances

Lambda Tuning:
‚Ä¢ User specifies desired closed-loop time constant (Œª)
‚Ä¢ Œª = how fast you want the loop to respond
‚Ä¢ Larger Œª = slower, more stable response
‚Ä¢ Recommended: Œª = 1.5 to 3 times process dead time
‚Ä¢ Formula: Kc = œÑ/(K(Œ∏+Œª)), Ti = œÑ

IMC (Internal Model Control):
‚Ä¢ Model-based tuning approach
‚Ä¢ Requires accurate process model
‚Ä¢ Provides excellent setpoint tracking
‚Ä¢ Single tuning parameter (filter time constant)

Relay Auto-Tune:
‚Ä¢ Automated method using on-off relay
‚Ä¢ Finds ultimate gain/period automatically
‚Ä¢ Common in modern smart controllers
‚Ä¢ Minimal process disturbance
Advanced Control Strategies ‚ñº
Cascade Control:
‚Ä¢ Primary (outer) loop controls main process variable
‚Ä¢ Secondary (inner) loop controls manipulated variable
‚Ä¢ Secondary loop must be 3-5x faster than primary
‚Ä¢ Example: Temperature (primary) cascading to steam flow (secondary)
‚Ä¢ Advantages: Rejects disturbances faster, improves stability

Feedforward Control:
‚Ä¢ Measures disturbance and compensates before it affects PV
‚Ä¢ Open-loop control (no feedback from process)
‚Ä¢ Usually combined with feedback (FF + FB)
‚Ä¢ Example: Compensating heat exchanger for inlet temp changes
‚Ä¢ Requires accurate model of disturbance effect

Ratio Control:
‚Ä¢ Maintains fixed ratio between two flows
‚Ä¢ Wild flow (uncontrolled) and captive flow (controlled)
‚Ä¢ Example: Air/fuel ratio in combustion
‚Ä¢ Ratio = Captive Flow / Wild Flow

Split Range Control:
‚Ä¢ Single controller output operates two final elements
‚Ä¢ Example: 4-12mA opens cooling valve, 12-20mA opens heating valve
‚Ä¢ Requires proper valve characterization and overlap

Override/Selective Control:
‚Ä¢ Multiple controllers, highest/lowest wins
‚Ä¢ Ensures constraints not violated
‚Ä¢ Example: High/low selector for temperature limits

Model Predictive Control (MPC):
‚Ä¢ Advanced multivariable control
‚Ä¢ Predicts future process behavior
‚Ä¢ Handles constraints and interactions
‚Ä¢ Used in complex processes (refineries, chemical plants)

Six Steps of LOTO:
1. Preparation: Identify all energy sources
   ‚Ä¢ Electrical, pneumatic, hydraulic, thermal, chemical
   ‚Ä¢ Review P&IDs and electrical drawings

2. Notification: Inform affected employees
   ‚Ä¢ Notify operators before shutdown
   ‚Ä¢ Post signs at control stations

   ‚Ä¢ Close valves, turn off switches

   ‚Ä¢ Tags identify person and reason

   ‚Ä¢ Attempt to start equipment
   ‚Ä¢ Use voltmeter, pressure gauge
